Settings & GitHub sync
github personal access token

This token lets every Save button push directly to GitHub. Stored only in your browser — never sent anywhere except GitHub's API directly.

To get a token: github.com -> Profile -> Settings -> Developer settings -> Personal access tokens -> Tokens (classic) -> Generate new token -> check repo -> no expiration -> copy it (starts with ghp_).

Once set, every Save button updates GitHub automatically. No Claude needed for logging.

anthropic api key (ai coaching)

Powers the AI coaching buttons — daily plans, progression recommendations, week guidance. Optional.

To get a key: console.anthropic.com -> API Keys -> Create Key -> copy it. ~$0.003 per button press.
