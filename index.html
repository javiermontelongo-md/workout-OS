github personal access token

All Save buttons write only to data.json. The UI file (index.html) is never touched during saves. You can redesign, recolor, or add fields to index.html at any time without losing any logged data.

To get a token: github.com -- Profile -- Settings -- Developer settings -- Personal access tokens -- Tokens (classic) -- Generate new token -- check repo -- no expiration -- copy it (starts with ghp_).

Stored only in your browser locally.
